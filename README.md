# 🦈 Wireshark HTTP Traffic Analysis Lab

## 📌 Overview

This project demonstrates hands-on experience using Wireshark to analyze network traffic and extract meaningful data from a PCAP file.

The lab focuses on inspecting HTTP traffic, following streams, and identifying hidden content within captured packets.

---

## 🛠️ Tools Used

* Wireshark
* Linux Terminal
* TryHackMe Lab Environment

---

## 🔍 Key Skills Demonstrated

* Packet inspection and filtering
* HTTP stream reconstruction
* Extracting files from PCAP
* Identifying encoded/compressed responses
* Analyzing web server responses

---

## 🧪 Tasks Performed

### 1. Applied HTTP Filter

Filtered traffic using:

```
http
```

---

### 2. Followed HTTP Streams

* Identified GET and HTTP responses
* Reconstructed full web pages from captured traffic

---

### 3. Extracted Hidden File

* Located `.txt` file within the capture
* Exported using:

```
File → Export Objects → HTTP
```

---

### 4. Analyzed Web Content

* Identified total number of artists
* Extracted artist names from HTML response

---

## 🎯 Key Findings

* Total artists: **3**
* Second artist: **The universe**
